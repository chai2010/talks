
幻灯片使用说明

# 安装 Go 语言包 和 git 工具

# 安装 Go语言扩展包的 present 工具
# 命令: go get golang.org/x/tools/present
# 文档: https://godoc.org/golang.org/x/tools/present
# 默认安装路径: HOME 目录下的 go/bin 目录

# 命令行切换到当前目录, 运行 present 命令
# 浏览器打开 http://127.0.0.1:3999

# 对外提供服务, 关闭 play 功能(安全措施)
# present -http=":3999" -play=false
